AnimeSaga is a fully responsive, modern web template designed for an anime streaming or information site. It focuses on a dark, aesthetic design with excellent readability and a comprehensive set of pages for managing content like ongoing series, upcoming releases, and genres.

This project is built using vanilla HTML, CSS, and JavaScript, heavily leveraging Tailwind CSS for rapid and clean styling.

# ‚ú® Features

Modern, Dark UI: Designed with a sleek, dark aesthetic using a contrasting yellow accent color for a premium feel.

Fully Responsive: Optimized for viewing on all devices (mobile, tablet, and desktop).

Modular Page Structure: Includes dedicated pages for major sections:

Home (index.html): Features carousels, featured shows, and a "Most Viewed" tab system.

Ongoing Series (ongoing.html): List of currently airing anime.

Upcoming Releases (upcoming.html): List of future anime seasons and titles.

Release Schedule (schedule.html): Daily or weekly release overview.

Genres (genres.html): Browse anime by genre.

Interactive Elements: Features a functional Sign In Modal and Mobile Menu Toggle powered by JavaScript.

Custom Animations: Smooth, custom CSS animations for page transitions and section fade-in effects (see style.css).

Utility Pages: Includes a Contact Us form (contact.html), Sign Up page (sign up.html), and a DMCA policy page (DMCA.html).
